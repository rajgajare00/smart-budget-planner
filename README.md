# smart-budget-planner

## Quickstart

1. Backend:
   - Open terminal in `backend`
   - `npm install`
   - `npm start`
   - API at `http://localhost:4000/api/message` and `http://localhost:4000/api/echo`

2. Frontend:
   - Open `frontend/index.html` in browser (or use local server like `npx serve frontend`)
   - Click buttons to call backend.
